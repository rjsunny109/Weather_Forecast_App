# Weather Forecast App

A simple weather forecast application built using Python, Tkinter, and the OpenWeatherMap API.

## Overview

This application allows users to enter the name of a city and retrieve real-time weather information including temperature, humidity, and conditions. The user-friendly graphical interface is created using the Tkinter library in Python.

## Features

- Input field to enter the city name.
- "Get Weather" button to fetch weather information.
- Display of weather information including temperature, humidity, and conditions.
- Stylish and intuitive user interface.

## Prerequisites

Before running the application, ensure you have the following:

- Python installed (Python 3.6 or later).
- An OpenWeatherMap API key. Sign up at https://openweathermap.org/api to obtain your API key.
